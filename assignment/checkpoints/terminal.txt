# 8 - Why would you run the command 'git status'?
To determine the status of your files, if they are clear or if your files have been staged for change and need to be committed and pushed to github
# 9 - What does the command 'git add .' do?
This command stages all files so that they are ready to be committed. 
# 10 - What is the command to push your committed changes to your repo on GitHub?
